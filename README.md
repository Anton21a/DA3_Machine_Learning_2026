# DA3_Machine_Learning_2026

This repository contains projects within the course of Data Analysis 3 Machine Learning. The work is divided into two main folders, each of them has a strict structure in the format: data/code/report. All data folders inside each assignment folder include raw and pre-processed data. Code chunks are located in 'code' folder and have been written with using R language. Reports are done using Latex software system, which allows to create a neat work with a standardized format. 

Assignment 1 contains the project on building prediction models on house prices in Budapest using different methods of machine learning. Whereas the second assignment focuses on finding fast growing firms using classification methods and loss function.
